# 기본 사용법

마크다운으로 문서를 작성하는 기본적인 방법을 알아봅니다.

## 텍스트 서식

**굵은 텍스트**, *기울임 텍스트*, ~~취소선~~ 등을 사용할 수 있습니다.

## 코드 블록

인라인 코드는 `backtick`으로 감싸고, 코드 블록은 세 개의 backtick으로 감쌉니다.

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```

## 리스트

### 순서 없는 리스트

- 항목 1
- 항목 2
  - 하위 항목 2-1
  - 하위 항목 2-2
- 항목 3

### 순서 있는 리스트

1. 첫 번째
2. 두 번째
3. 세 번째
