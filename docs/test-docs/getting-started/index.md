# 시작하기

Dev Book에 오신 것을 환영합니다. 이 문서는 프로젝트를 빠르게 시작하는 방법을 안내합니다.

## 개요

Dev Book은 마크다운 파일을 기반으로 한 문서 웹사이트입니다. GitBook과 유사한 인터페이스를 제공하며, 온프레미스 환경에서 운영할 수 있습니다.

## 주요 기능

- 마크다운 파일을 웹페이지로 변환
- 자동 사이드바 생성
- 문서 내 목차(TOC) 자동 생성
- 코드 하이라이팅 지원
